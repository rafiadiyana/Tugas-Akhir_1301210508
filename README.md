# Tugas-Akhir_1301210508
Judul: STUDI KOMPARASI WORD EMBEDDING PADA MODEL LSTM UNTUK ANALISIS SENTIMEN ISU KDRT DI TWITTER
Abstrak:
Kekerasan dalam rumah tangga (KDRT) merupakan isu sosial yang mendominasi kasus kekerasan berbasis gender di Indonesia, dengan 336.804 kasus atau 99% dari total 339.782 kasus yang dilaporkan pada Catatan Tahunan Komnas Perempuan tahun 2023. Melimpahnya data percakapan publik terkait isu KDRT di Twitter memungkinkan dilakukannya analisis sentimen untuk memahami pola persepsi dan respons masyarakat. Penelitian ini melakukan studi komparasi terhadap tiga metode word embedding, yaitu Word2Vec, GloVe, dan FastText, pada model Long Short-Term Memory (LSTM) untuk analisis sentimen isu KDRT di Indonesia pada platform Twitter. Ketiga embedding dilatih secara mandiri pada korpus Wikipedia Indonesia dengan dimensi vektor 300 dan epoch 10. Dataset terdiri dari 3.223 tweet yang terbagi menjadi 672 data sentimen positif dan 2.551 data sentimen negatif. Pengujian dilakukan dalam dua skenario, yaitu tanpa penanganan ketidakseimbangan data dan dengan penerapan class weight. Pada skenario pertama, LSTM dengan Word2Vec mencatatkan performa terbaik dengan accuracy 0.9054 dan F1-Score 0.8423. Pada skenario kedua, LSTM dengan GloVe mencatatkan performa terbaik dengan accuracy 0.9054 dan F1-Score 0.8454. Secara keseluruhan, GloVe merupakan metode word embedding yang memberikan performa terbaik dalam penelitian ini ketika dikombinasikan dengan model LSTM dan penanganan ketidakseimbangan data menggunakan class weight.
Kata Kunci: KDRT, analisis sentimen, word embedding, Word2Vec, GloVe, FastText, LSTM, class weight
